Formação

Sou Gustavo, apaixonado por dados e pelo impacto que eles podem gerar. Minha formação em Ciências de Dados pela FIAP e em Tecnologia da Informação pela UNIVESP, aliada a certificações renomadas como Analista de Dados (Google) e Engenharia de Dados (IBM), reforça minha expertise técnica.

Habilidades

Domino ferramentas como Excel e Power BI para criar insights visuais impactantes, utilizo SQL e Python para manipulação e análise de dados avançada, e aplico conceitos de Estatística para decisões orientadas por evidências. Tenho experiência em HTML, CSS e JavaScript para desenvolvimento web, além de competências em modelagem e arquitetura de dados, essenciais para estruturar informações de forma estratégica.
